How to Use

  1. Copy a column from Excel (Ctrl+C)
  2. Paste into the input box above (Ctrl+V) - converts automatically
  3. Select your format: ; or ;=
  4. Click Copy to Clipboard to use the result
  5. Use Semicolon Equals for exact matches on part numbers
  6. Don't copy more than 2500 parts
